面向对象:创建对象-原型-原型链-继承

发表于 2016-03-12 |

创建对象的方式

  1. 工厂模式

    • 语法

    • 说明

      1. 在函数中定义对象,并定义对象的各种属性
      2. 创建对象时，不需要new操作符
      3. 在函数的最后返回该对象

  2. 构造函数

    • 语法

    • 说明

      1. 无需再函数内部创建对象
      2. 使用this给当前实例添加成员
      3. 函数无需明确return

  3. 原型模式

    • 语法

    • 说明

      1. 函数中不对成员进行定义
      2. 利用prototype属性对成员进行定义

  4. 混合模式

    • 语法

    • 说明

      1. 该模式是指混合搭配使用构造函数方式和原型方式
      2. 将所有属性定义在函数中（构造函数方式）
      3. 将所有方法定义在原型上
      4. 推荐使用这样方式创建对象

原型的概念

  1. 概念：所谓原型就是指 函数的prototype属性所引用的对象

    • 只要定义一个函数，那么就会创建其原型
    • 由上述函数创建出来的所有对象，都可以直接访问到原型上的成员

  2. 作用：实现同类对象之间的数据共享。

  3. 获取原型方式

    • 通过函数：.prototype
    • 通过对象：.__proto__
    • 注意：已__开头和结尾的属性均为非标准属性; 
      原型 默认具有constructor属性，引用的就是其构造函数

发表于 2015-06-26 |
  1. HTML精确定位:scrollLeft,scrollWidth,clientWidth,offsetWidth
  2. scrollHeight: 获取对象的滚动高度。
  3. scrollLeft:设置或获取位于对象左边界和窗口中目前可见内容的最左端之间的距离

  7. offsetLeft:获取对象相对于版面或由 offsetParent 属性指定的父坐标的计算左侧位置
